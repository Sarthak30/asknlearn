Knowledge Pool


About

The Internet has risen as fastest, affordable, convenient and reliable source of information transfer and communication over the past few years. An Internet-based web portal has emerged an essential role in information distribution among people because of the convenience and faster reach. The Knowledge Pool is an online web platform where both students and teachers can interact with each other outside the classrooms to share educational and academic information. They can post links to related videos, lectures, .PDF, .PPT etc files through their respective login accounts. The users can post their queries for help and share their opinions about any particular post.Knowledge Pool primarily focuses on narrowing the communication gap between teachers and students to improve education quality.

Features

  • Main Web Application
    • Knowledge pool basically targets to reduce the gap between teacher and student interaction outside the classrom with the help of of an online web application.
  • Summarization
    • Unlike other web applications like twitter, facebook which shows you the whole post conatining a bunch of useless lines ,, this web application shows you the abstract content of the posts of the ones whom you follow.
  • Email and Sms
    • This application is embeeded with the feature of direct mailing and texting to the active users.
  • File Uploads
    • Any user can upload/download any file that is on the server according to access level provided by the admin.
  • Media Library
    • Any user can upload file in a variety of formats like .pdf, any media file, .ppt, .doc and many more.
  • Bookmarks
    • Teachers can upload their research papers, their works of intrest , their publications etc.
  • Chat Application
    • Student can get in touch with the teacher with the help of chat feature that is embeeded in the application
